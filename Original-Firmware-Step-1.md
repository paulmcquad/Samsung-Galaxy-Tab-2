# Original Firmware on Samsung GT-P3110

## Samsung Galaxy Tab 2 7.0

Android Version : 4.2.2 (Jelly Bean)

## Enter Recovery Mode

1. Turn Off:
2. Press Power Button + Volume up Key
3. Android System Recovery

## Enter Custom OS Warning Menu

1. Turn Off:
2. Press Power Button + Volume down Key
3. Custom OS Warning Menu Appears
